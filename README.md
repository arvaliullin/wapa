# WAPA

WebAssembly Performance Analyzer (WAPA) — это специализированное программное обеспечение,
предназначенное для выполнения модулей WebAssembly (wasm) в различных средах, включая Raspberry Pi (RPI),
настольные компьютеры и другие устройства. WAPA позволяет измерять время, необходимое для выполнения этих модулей,
и хранить данные для проведения сравнительного анализа.
Для корректного функционирования WebAssembly-модули должны соответствовать определённому интерфейсу платформы.
WAPA - платформа для проведения экспериментов по замеру времени выполнения wasm - модулей.

## WapaComposer

WapaComposer - это сервис, который предоставляет restful api для взаимодействия с пользовательским интерфейсом.
Задачи WapaComposer:
1. Представление результатов измерений.
2. Формирование заданий для выполнения замеров времени выполнения.
3. Сохранение результатов выполнения замеров времени исполнителями.

Алгоритм формирования заданий для выполнения замеров времени выполнения:
1. получения wasm-модулей,
2. публикации их на ftp-сервере и формировании информации о задании в базе данных,
3. публикации сообщения в очередь performance_test для дальнейшего тестирования.

ЯП: Golang

## WapaRunner


WapaRunner - сервис который 
1. получает сообщения об обработки из очереди performance_test
2. получает wasm-модуль с ftp
3. выполняет замеры времени выполнения для модуля
4. публикует результаты в очередь analyzer_result

PerformanceAnalyzer  - может быть rpiPerfomaceAnalyzer, x86_64PerformanceAnalyzer, BrowserPermanceAnalyzer.

Реализуется на Js.

## Используемое ПО
1. Очередь сообщений -  NATS
2. База данных - postgresql
